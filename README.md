# HouseZone – House Management System

## 📌 Overview

HouseZone is a web-based House Management System developed to simplify the process of renting and selling houses. Users can browse available properties, view detailed house information, create an account, and submit booking payment details.

The platform also includes an admin panel where administrators can manage property listings by adding, updating, or removing houses. All data is securely stored in a MySQL database, ensuring efficient and reliable property management.

---

## 🚀 Features

* User registration and login
* Browse and view property details
* Submit booking payment details
* Admin dashboard to add, update, and delete properties

---

## 🏗️ Tech Stack

* **Frontend:** ReactJS, CSS
* **Backend:** Spring Boot (Java)
* **Database:** MySQL
* **IDE:** IntelliJ IDEA

## ⚙️ Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/PiyumiSandunika/HouseZone.git
cd housezone
```

### 2️⃣ Backend Setup

* Open the backend project in IntelliJ IDEA
* Configure `application.properties` with your MySQL credentials
* Run the Spring Boot application

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```
